# CLI (Command Line Interface)

`CLI 는 기본적으로 사용하던 GUI 방식이 아닌 Command 를 사용한 조작법`

- `pwd(Print working directory)` : **현 위치 확인**

- `cd(Change directory)` : **폴더변경** 
  - cd 폴더이름
- `ls(list)` : **현재 폴더 리스트**
- `mkdir(Make Directory)` : **폴더 생성**
  - `mkdir 폴더 이름`
- `touch` : **파일생성**
  - `touch 파일이름.형식`
- `rm(Remove)` : **지우다(파일 삭제)**
  - `rm 파일이름`
- `rm -r` : **지우다(폴더 삭제)**
  - `rm -r 폴더 이름`

> 이 문서는 용어만 다루고 있음.
